# RoomEscapeGame
2021 유니티팀 박해지

 
&#10024; 게임소개
------------------
아들을 잃은 한 병원의 원장이 아들을 닮은 아이를 불러들여 영혼교체 실험을 하였고  
어떤  **사건**이 있음직후로부터 10년 후, 악몽에서 깬 주인공은 그 저택으로 향하게 되고   
그 때의 사건의 진실을 알아가면서 저택을 탈출해나가는 방탈출 게임 **(데모버전)**

&#10024; 조작법
------------------
* 인벤토리 열기/닫기 : I
* 노트 창 끄기 : Q
* 아이템 사용: 우클릭
* 보기 : 좌클릭
